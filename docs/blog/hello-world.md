Welcome to nightd! We're building a daemon to schedule autonomous coding agents.

## Why nightd?

As AI coding assistants become more capable, we need reliable infrastructure to run them autonomously. nightd fills this gap by providing:

- Scheduled execution of coding tasks
- Resource management and monitoring
- A simple REST API for integration
- Lightweight, reliable daemon architecture
